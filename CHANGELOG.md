# Changelog

All notable changes to the Playwright Browser Power will be documented in this file.

## [2.0.0] - 2026-05-01

### Added
- **Comprehensive Installation Guide** - New INSTALL.md with platform-specific instructions
- **Quick Start Examples** - Four ready-to-use examples in POWER.md and README
- **When to Use Section** - Clear guidance on ideal use cases and alternatives
- **Framework Integration** - Setup instructions for React, Vue, Svelte, Angular, and plain HTML
- **Common Issues & Solutions** - Expanded troubleshooting with 8+ common scenarios
- **Success Indicators** - Clear metrics to know the power is working correctly
- **Power Combinations** - Shows how to use with Figma, Stripe, Supabase, and other powers
- **Agent Instructions** - Detailed workflow guide for AI agents using this power
- **Tool Selection Guide** - Table showing when to use each browser tool
- **Workflow Decision Tree** - Helps choose the right steering file for each task
- **Error Recovery Guide** - Step-by-step recovery for common errors
- **Platform-Specific Notes** - Detailed setup for Windows, macOS, Linux, Docker/CI

### Changed
- **Refined Keywords** - Removed overly broad keywords to prevent false activations
- **Improved Keyword Specificity** - Focused on unique, power-specific terms
- **Enhanced POWER.md Structure** - Better organization with clear sections
- **Better README Flow** - More logical progression from installation to usage
- **Clearer Troubleshooting** - Cause → Solution format for all issues

### Improved
- **Documentation Quality** - More examples, clearer explanations
- **User Onboarding** - Smoother first-time experience
- **Agent Guidance** - Better instructions for AI agents
- **Error Messages** - More helpful error descriptions
- **Configuration Options** - Clearer customization instructions

### Fixed
- Removed generic keywords that caused false activations
- Clarified browser installation process
- Improved error recovery instructions
- Better platform-specific guidance
